# Digital Aadhaar – Cybersecurity Awareness Platform

Digital Aadhaar is an interactive cybersecurity awareness platform designed to help users safely learn digital tasks while identifying online scams, phishing attempts, deepfakes, QR frauds, and other cyber threats in a zero-risk environment.

## Features
- Scam chat simulation
- AI-powered scam interaction training
- QR fraud detection
- Deepfake recognition lab
- Community-based scam alerts
- Multilingual support
- Progress tracking and gamification
- Voice-guided accessibility support

## Modules
- Learn
- Scam Detector
- Scam Chat
- Deepfake Recognition
- Community Siren
- Progress Tracker

## Tech Stack
- Ionic
- React
- TypeScript
- Firebase
- Express.js

## Future Scope
- Voice scam detection
- Advanced deepfake detection
- Government platform integration
- Nationwide scam alert network
